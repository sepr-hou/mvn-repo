# SEPR-HOU Maven Repo
Contains maven jars which are not in any central repository yet.
Currently only contains kryonet.

## Usage
To use add the following repository to your pom.xml

    <repositories>
      <repository>
        <id>sepr-hou</id>
        <url>https://raw.github.com/sepr-hou/mvn-repo/master/</url>
      </repository>
    </repositories>
